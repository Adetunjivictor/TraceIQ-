# 👁 TraceIQ — Wallet Intelligence Bot
Built by @its_vicex

## Commands
- `/scan` — Analyze any wallet (Solana, ETH, BNB)
- `/pnl` — Upload a PNL card image to extract & analyze wallet
- `/top` — Find top 80-100% win rate wallets from a token contract
- `/dev` — Analyze the deployer wallet of any token

---

## 🚀 Deploy on Railway (iPhone-friendly)

### Step 1 — Upload to GitHub
1. Go to **github.com** on your phone browser
2. Create a free account
3. Tap **+** → **New repository** → name it `traceiq` → Create
4. Tap **uploading an existing file**
5. Upload ALL files in this folder (bot.py, config.py, requirements.txt, Procfile, railway.toml, and the modules folder)

### Step 2 — Deploy on Railway
1. Go to **railway.app** → Sign in with GitHub
2. Tap **New Project** → **Deploy from GitHub repo**
3. Select your `traceiq` repo
4. Railway auto-detects Python and installs everything
5. Your bot goes live in ~2 minutes ✅

### Step 3 — Verify
Open Telegram → find your bot → send `/start`
If it responds, you're live! 🎉
